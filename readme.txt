
1.  data文件夹：存储原始音频数据，以“x-y.wav”形式命名，1<=x<=10，1<=y<=5。

    其中x(1~10)分别表示“开门、关门、开灯、关灯、开空调、关空调、开电视、关电视、烧水、蒸饭”这十个词。

    y(1~5)表示一个词录制五遍。

    且“x-1.wav”（1<=x<=10）为模板文件，其余为测试文件。


2.  pcm文件夹：存储data文件夹下原始数据对应的端点检测后的文件。


3.  mfc文件夹：存储pcm文件夹下的数据经HTK工具包特征提取后的文件。


4.  online/data：存储实时录制的文件。

    online/wavedection：存储录制的原始文件经端点检测后的文件。


5.  实验报告：实验报告的word版和pdf版。


6.  main.py：主模块文件，可直接运行。
